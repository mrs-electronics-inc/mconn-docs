### How do I change the Ethernet Mode?

The Ethernet mode can be switched between DHCP Server and Client using the `eth-conf.sh` utility — for full details, see [Switching Ethernet Modes](../../../getting-started/setting-up-mconn#switching-ethernet-modes).
