# MConn Module Guide
MConn Qt Application Development refers to the beginning phase of creating an application intended for the MConn platform using Qt. This widely used cross-platform framework enables developers to create applications with native-like user interfaces.

- **CAN Module**: Handles sending and receiving messages over the CAN bus.
- **IO Module**: Controls and reads GPIO and analog/digital inputs, digital outputs , Buzzer and Screen brightness.
- **Camera Module**: Streams video from connected camera devices.
- **Video Player Module**: Plays local video files.
- **Audio Player Module**: Plays audio with volume and mute control.
